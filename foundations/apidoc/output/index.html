      Query for an existing block(s) based on timestamp
      Request:
      Resource:
      http://uofmblockchainanalytics.com/bitcoin/blocks
      Query String Parameters:
      Parameter Type Optional Description
      date_offset integer Timeframe of the required data
      day integer Day of block creation
      month integer Month of block creation
      year integer Year of block creation
      Body:
      {
      "date_offset": integer, Timeframe of the required data
      "day": integer, Day of block creation
      "month": integer, Month of block creation
      "year": integer Year of block creation
      }
      Response:
      Body:
      {
      "Block": Block
      }
      Codes:
      Code Message Description
      404 Block not found Block does not exists

      Request:
